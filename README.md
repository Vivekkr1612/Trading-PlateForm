# Trading Platform (Zerodha Clone)

## Overview

Trading Platform is a full-stack web application inspired by the Zerodha trading platform. It is built using the MERN stack and demonstrates authentication, portfolio management, holdings, positions, and a responsive dashboard.

## Features

* User Authentication
* Dashboard
* Holdings Management
* Positions Management
* Watchlist
* Responsive UI
* REST API Integration
* MongoDB Atlas Database

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* Passport.js
* Bootstrap

## Folder Structure

```
Trading Clone
│
├── Backend
├── Dashboard
└── frontend
```

## Installation

```bash
git clone https://github.com/Vivekkr1612/Trading-PlateForm.git

cd Backend
npm install
npm start

cd ../frontend
npm install
npm start

cd ../Dashboard
npm install
npm start
```
